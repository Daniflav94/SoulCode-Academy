Comandos HTML


Tags estruturais:

< html > - Informa ao navegador que será utilizado o HTML para marcação
< head > - Informações não visiveis na página
< meta name = "" content = "" > - Meta Tags: informações sobre a página
< /head >
< body > - Corpo da página
< h1 > < /h1 > (até h6) - Níveis de título, de relevância
< p > < /p > - Parágrafo
< /body >
< /html >
